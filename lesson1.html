Задание 2.2¶

Решите систему уравнений:

Линейная или нелинейная это система?
А каждое уравнение по отдельности?

Решение¶

Данная система уравнений является не линейной.
Первое уравнение системы - квадратное.
Выразим из второго уравнения y через x и подставим в первое

Теперь, подставив x в любое уравнение, найдём y

Проверка¶

In [3]:
from scipy.optimize import fsolve

def equations(p):
    x, y = p
    return (x**2 + y*x - 9, x -y/5)

fsolve(equations, (1, 1))
Out[3]:
array([1.22474487, 6.12372436])

Задание 3¶

Решите задачу:
Площадь пола прямоугольной комнаты равна 48 м2,а его периметр равен 28 м. Найдите длину и ширину комнаты.

Решение¶

Составляем систему уравнений:

Выразим из первого уравнения a через b и подставим во второе уравнение:

Ответ: Длина - 8м, ширина - 6 м.

Проверка¶

In [4]:
from scipy.optimize import fsolve

def equations(p):
    x, y = p
    return (x*y - 48, 2*(x + y) - 28)

fsolve(equations, (1, 1))
Out[4]:
array([8., 6.])

Решение¶

import matplotlib.pyplot as plt
from matplotlib import style

style.use('fivethirtyeight')

x = np.linspace(-2 * np.pi, 2 * np.pi, 1024)

k = 1
plt.figure(figsize=(8, 6))
plt.plot(x, np.cos(k*x))
k -= 0.5
plt.plot(x, np.cos(k*x))
plt.xlabel('x')
plt.ylabel('y')
plt.grid(True)
